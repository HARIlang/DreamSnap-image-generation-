# 🌌 DreamSnap – Prompt to Image Generator

Turn your **imagination into stunning visuals** with just a few words! ✨  
DreamSnap lets you enter a prompt and watch AI bring your ideas to life instantly.  

---

## 🚀 Live Demo  
🔗 [**Try DreamSnap Now**](https://lnkd.in/gn9NMsHG)  

---

## 🖼 Features
- 🎨 **Prompt to Image** – Enter text, get AI-generated images in seconds.  
- ⚡ **Real-time Generation** – Powered by **ClipDrop Text-to-Image API**.  
- 📱 **Responsive Design** – Works seamlessly on desktop & mobile.  
- 🧭 **Smooth Navigation** – Built with **React Router DOM**.  
- 🦴 **Skeleton Loader** – Better UX while images load.  

---

## 🛠 Tech Stack
- **Frontend**: React, Tailwind CSS  
- **Routing**: React Router DOM  
- **Loading Effects**: React Skeleton Loader  
- **AI Integration**: ClipDrop Text-to-Image API  

---

## 📸 How It Works
1. Type your **creative prompt**.  
2. Click **Generate**.  
3. Let AI transform your words into **beautiful artwork**.  
4. Download & share your creation!
